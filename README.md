## ✨ À propos

Portfolio personnel de **Steve Donald Compaoré**, étudiant en 3ème année informatique à l'Université de Tokat Gaziosmanpaşa, Turquie. Ce site présente mes projets, compétences, parcours et ambitions — notamment celle de construire une startup tech africaine de classe mondiale.

Le portfolio est disponible en **3 langues** : Français 🇫🇷, English 🇬🇧 et Türkçe 🇹🇷.

---

## 🎯 Sections

- **Hero** — Introduction animée avec effet de typing et particles
- **À propos** — Parcours, ambitions et valeurs
- **Projets** — Portfolio de projets avec cards 3D tiltables et liens live/GitHub
- **Compétences** — Stack technique visualisée par catégories
- **Expérience** — Timeline académique et professionnelle
- **Langues** — Niveaux en FR, EN, TR avec progress bars animées
- **AI System** — Section dédiée à mon intégration de l'IA dans mes projets
- **Contact** — Formulaire de contact et liens sociaux

## 🚀 Installation locale

```bash
# 1. Cloner le projet
git clone https://github.com/dosteeve2-hash/steeve-portfolio.git
cd steeve-portfolio

# 2. Installer les dépendances
npm install

# 3. Lancer en développement
npm run dev
# → http://localhost:3000
```

Aucune variable d'environnement requise — le portfolio est entièrement statique.

---

## 🌍 Internationalisation

Le portfolio supporte 3 langues via `next-intl` :

```
/        → Redirection automatique selon le navigateur
/fr      → Français (langue principale)
/en      → English
/tr      → Türkçe
```

Les traductions sont dans `messages/` : `fr.json`, `en.json`, `tr.json`.
